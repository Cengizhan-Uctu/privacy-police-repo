Privacy Policy

This Chrome extension is designed to help users maintain browser performance by clearing cached data if Chrome has been running continuously for 3 consecutive days.

The extension does not collect, store, or transmit any personally identifiable information (PII). It does not use or access external servers or third-party services.

The extension only uses local browser APIs such as alarms, tabs, storage, and browsingData to perform its intended functionality. All operations happen within the user's browser environment.

No user data is tracked, logged, shared, or sold. We respect your privacy.

If you have any questions about this privacy policy, feel free to contact us via the email provided in the Chrome Web Store listing.
